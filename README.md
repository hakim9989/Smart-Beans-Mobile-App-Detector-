 Bean Mobile App Detector (Flutter + TensorFlow Lite)

This project is a Flutter-based mobile application that performs real-time object detection using an embedded TensorFlow Lite machine learning model. The model was trained on a custom dataset and deployed directly on-device for offline inference.

 Key Features

- Real-time object detection using camera
- On-device inference using TensorFlow Lite (no internet required)
- Fully offline functionality
- Cross-platform Flutter implementation
- Integrated custom-trained ML model

 Machine Learning Pipeline

1. Dataset Collection and Preparation
2. Model Training using TensorFlow
3. Model Conversion to TensorFlow Lite (.tflite)
4. Integration into Flutter application
5. Real-time inference using device camera

 Technologies Used

- Flutter (Dart)
- TensorFlow
- TensorFlow Lite
- Python (for model training)
- Mobile Camera Integration

 Project Architecture

- Flutter frontend
- Embedded TensorFlow Lite model
- On-device inference engine
- No external API dependency
